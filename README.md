# Reborn50

这是一款让你爱上自律的小游戏

## Reborn50 Introduction

Reborn 灵感诞生于经典的 Project50 自律挑战，我们希望为这场自我蜕变之旅锦上添花。在这里，你既能循序渐进养成自律、健康的生活与学习习惯，完成人生重启、实现自我成长，也能告别枯燥打卡，感受挑战的趣味与乐趣。最终让你主动拥抱好习惯，让自律成为生活常态。

## PROJECT50 Introduction

PROJECT50  是一项生活方式挑战，专为未来顶尖的1%人群打造。

无论你是创意从业者、运动员、创业者、精英人士，还是只想全力以赴活出精彩，这个挑战都适合你。

### PROJECT50 挑战七大核心准则

- **RULE 1: 早上8点前起床**:保证每日睡眠不少于6小时，作息规律，可根据自身情况灵活调整。
- **RULE 2: 完成晨间流程**:留出一小时不受外界干扰。远离手机与社交软件，时间充裕时可冥想、散步、深度思考，为一天做好规划。
- **RULE 3: 每日运动一小时**:根据自身体能选择运动方式，散步、拉伸、跑步、瑜伽、骑行、健身等均可。
- **RULE 4: 每日阅读10页书籍**:阅读励志、心理学、商业类或各类知识性书籍。
- **RULE 5: 每日花一小时学习新技能**:挑战开始前提前规划目标与想要学习的技能，学习时专心致志、不受打扰。
- **RULE 6: 坚持健康饮食**:选择适合自己的饮食方案，戒酒、不喝碳酸饮料、减少糖分摄入，同时保证充足饮水。
- **RULE 7: 用手账本记录每日进度**:记录当日得失、所学感悟以及可以改进的地方。

### 项目通用规则

在50天挑战期间，只要有一天未能遵守规则，就必须**从第1天重新开始**，直至完整完成挑战。
